# Streamlit-приложение для анализа тональности

## Заметки по проекту

- Основной файл приложения: `emotion.py`
- Приложение написано в одном файле, как и требовалось
- Поддерживается ручной ввод текста
- Поддерживается загрузка одного или нескольких файлов `.txt`, `.md`, `.csv` и `.json`
- Если загружены файлы, для анализа используется текст из файлов, а не из поля ввода
- Для анализа тональности используется русскоязычная модель HuggingFace Transformers `blanchefort/rubert-base-cased-sentiment`
- Дополнительно используются zero-shot модели Transformers для определения эмоций и тем
- В интерфейсе показываются тональность, эмоции, темы и важные фрагменты текста

## Внесенные изменения

- Создан Streamlit-интерфейс с кнопкой `Analyze text`
- Добавлено извлечение текста из загруженных файлов
- Реализован анализ нескольких файлов за один запуск
- Реализован приоритет содержимого файлов над текстовым полем
- Добавлен модульный код с отдельными функциями для:
  - загрузки пайплайна
  - извлечения текста
  - выбора источника текста
  - анализа тональности
- определения эмоций
- определения тем
- поиска и подсветки важных фрагментов
- вывода результата
- Добавлена обработка ошибок для пустого текста, неверного формата файла и проблем декодирования
- Файл `app.py` переименован в `emotion.py`
- Добавлен `run_emotion.bat` для быстрого запуска приложения в Windows
- Заменена стандартная sentiment-модель на русскоязычную, чтобы корректнее анализировать отзывы на русском языке

## Локальный запуск

Установка зависимостей:

```bash
"C:\Users\user\AppData\Local\Programs\Python\Python314\python.exe" -m pip install streamlit transformers torch
```

Запуск приложения напрямую:

```bash
"C:\Users\user\AppData\Local\Programs\Python\Python314\python.exe" -m streamlit run emotion.py
```

Быстрый запуск через bat-файл:

```bash
run_emotion.bat
```
